# Unidade 4: Comandos de Controle de Fluxo - Seleção

Os exercícios abaixo focam na estrutura de **Seleção Simples (if)**, utilizando diferentes operadores relacionais para validar condições específicas.

---

## 📝 Lista de Exercícios Complementares: Seleção Simples

### Uni4Comp01.java
**Enunciado:** Uma loja virtual deseja dar um brinde para categorias específicas de clientes. Escreva um programa que leia um caractere representando o tipo de cupom ('S' para Especial, 'C' para Comum). Se o cupom for **igual** ('==') a **'S'**, o programa deve exibir a mensagem: *"Cupom validado! Você ganhou um adesivo exclusivo."*

| Exemplo de entrada | Exemplo de saída |
| :--- | :--- |
| Tipo de cupom: S | Cupom validado! Você ganhou um adesivo exclusivo. |
| Tipo de cupom: C | (sem saída) |

---

### Uni4Comp02.java
**Enunciado:** Um sistema de monitoramento de TI verifica o status de um servidor através de códigos numéricos. O código **1** significa que o sistema está "OK". Escreva um algoritmo que leia um número inteiro representando o status atual. Se o status for **diferente** ('!=') de **1**, o programa deve exibir o alerta: *"ATENÇÃO: Sistema operando com instabilidade!"*

| Exemplo de entrada | Exemplo de saída |
| :--- | :--- |
| Status: 2 | ATENÇÃO: Sistema operando com instabilidade! |
| Status: 1 | (sem saída) |

---

### Uni4Comp03.java
**Enunciado:** Uma transportadora utiliza uma esteira de separação rápida para pacotes leves, visando agilizar a logística. Escreva um algoritmo que leia o peso de uma caixa (em kg). Se o peso for **menor** ('<') que **2.5 kg**, o programa deve informar: *"Encaminhar para a Esteira Rápida (Pacote Leve)"*.

| Exemplo de entrada | Exemplo de saída |
| :--- | :--- |
| Peso: 1.8 | Encaminhar para a Esteira Rápida (Pacote Leve) |
| Peso: 5.0 | (sem saída) |

---

### Uni4Comp04.java
**Enunciado:** Um portal de jogos educativos possui uma área de chat restrita para adolescentes. Para acessar, o usuário deve ter uma idade mínima. Escreva um programa que leia a idade do usuário e, caso ela seja **maior ou igual** ('>=') a **13 anos**, exiba a mensagem: *"Acesso ao chat liberado!"*.

| Exemplo de entrada | Exemplo de saída |
| :--- | :--- |
| Idade: 13 | Acesso ao chat liberado! |
| Idade: 10 | (sem saída) |

### Uni4Comp05.java
**Enunciado:** Escreva um programa que valide um cupom de desconto em formato de texto. Leia o nome do cupom digitado pelo usuário. Se o cupom for exatamente igual a **"SOUFURB"**, o programa deve exibir a mensagem: *"Desconto de 15% aplicado com sucesso!"*.

| Exemplo de entrada | Exemplo de saída |
| :--- | :--- |
| Cupom: SOUFURB | Desconto de 15% aplicado com sucesso! |
| Cupom: PROMO10 | (sem saída) |
